Reportar Nivels

Los datos para esta aplicación provienen de una spreadsheet que contiene un base de datos storicos de las elevaciones para cada embalse. Puedes ver y actualizar los datos en esta spreadsheet con Google Sheets. Debe actualizar la información en esa spreadsheet de Google todos los días. Cuando inicie la aplicación, debe usar el botón en esta pagina para sincronizar los datos de Google con los datos que tiene este app.

Reporting Levels

The data in this app comes from a spreadsheet that contains a simple database of historical elevation levels for each of the reservoirs. You can view and update the data on that spreadsheet on Google Sheets. You should update the information on that Google Sheet every day. When you launch the app, you should use the synchronize data button available on this page to update the local database of information maintained by the app.
